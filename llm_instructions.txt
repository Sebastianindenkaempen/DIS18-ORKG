1. Downloading Ollama
-   visit https://ollama.com/download and download the installer
-   execute OllamaSetup.exe

Dowloading Mistral
-   open cmd
-   run the command "ollama" to check if the installation was successful
-   run the command "ollama run mistral" which will then download mistral
-   you can now prompt in cmd or access the LLM via VS Code

3. Ollama
-   if Ollama is not running, use the comman "ollama serve"
-   Ollama runs on http://localhost:11434
